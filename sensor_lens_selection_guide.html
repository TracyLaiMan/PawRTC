一、Sensor 选型 — 8 个关键维度

Sensor、ISP、镜头三者耦合。先定应用场景 → 再选 Sensor 规格 → 最后匹配镜头光学参数。

1

分辨率 (Resolution)

Resolution — 不是越高越好
使用场景推荐分辨率说明
家庭看护 / 门铃2MP (1920×1080)足够看清人脸
庭院 / 门口3~4MP兼顾细节和存储
商铺 / 停车场4~5MP需要车牌/人脸细节
大场景 / 周界8MP 或双摄拼接大范围覆盖

⚠️ 同样光学尺寸下，分辨率越高→单像素越小→感光能力越弱→夜视越差。2MP 是家用 IPC 甜点。

2

2.7μm★★★☆☆良好SC2337P / GC2083

1.45μm★☆☆☆☆差仅限光线充足的场景

每个像素都是「光子桶」。尺寸越大，收集的光子越多。SC231AI 的 2.8μm 比 SC2337P 的 2.7μm 多约 7% 受光面积，配合 BSI 工艺，感光度差距可达 24.8%。

核心结论

  • 画质王者：SC231AI — 感光度 7387+BSI+60fps，旗舰首选
  • 最佳性价比：SC223A — 感光度 7330（差 <1%），价格低约 50%
  • BSI 平衡之选：OS02G10 — BSI 工艺 + OmniVision 品牌，12 英寸晶圆供货稳
  • 星光入门：GC2053 — 感光度 3870，81dB DR，低价中画质尚可
  • 极致降本：GC2083 — 价格最低，仅适合有补光的环境
  • 不推荐：SC2337P — 被 SC223A 全面碾压（感光度差 24%、DR 低 4dB、像素更小）

三、镜头选型 — 6 个核心要素

镜头决定了有多少光能到达 Sensor，是画质链路的第一个瓶颈

1

焦距 (Focal Length)

Focal Length — 决定视角和监控距离
焦距水平视角
(1/2.9")		最佳监控距离场景
2.8mm~95°3~5m室内 / 门口 / 走廊
3.6mm~75°5~10m庭院 / 大厅
6mm~45°10~20m停车场 / 周界
8mm~33°20~30m出入口 / 闸机
12mm~22°30~50m远距离识别

FOV 计算公式：FOV = 2 × arctan( Sensor宽度 / (2 × 焦距) )

4

CRA 匹配 (Chief Ray Angle)

CRA Matching — 容易被忽略的关键参数

镜头的出射角 (CRA) 必须 ≤ Sensor 的主光线入射角 (CRA)，否则画面边缘明显偏暗。

SensorSensor CRA镜头 CRA 要求匹配结果
SC223A15°≤ 15°推荐 12~15° 镜头
OS02G1015°≤ 15°推荐 12~15° 镜头
SC2337P15°≤ 15°推荐 12~15° 镜头
GC205312°≤ 12°需要低 CRA 镜头
SC231AI未公开需向 FAE 确认建议实测

GC2053 的 CRA 只有 12°，镜头选择范围受限——需要专门的低 CRA 镜头。SC223A 的 15° CRA 兼容性更好。

5

IR-Cut 兼容性

IR-Cut Compatibility — 日夜切换的关键

日夜全彩 IPC 的核心功能。选镜头时需关注：

  • 共焦设计：IR-Cut 切换后，后焦不偏移，画面不虚焦
  • 红外镀膜：可见光 + 近红外波段 MTF 都要好
  • 推荐全玻璃镜头（All-Glass），高温不跑焦
6

镜片材质与镀膜

Lens Material — 影响温度稳定性和像质
材质特点推荐场景
全玻璃 (All-Glass)耐温好，不跑焦，寿命长室外 IPC（推荐）
玻璃+塑胶 (3G1P)成本低，高温可能跑焦室内 IPC
全塑胶成本最低，易跑焦不推荐安防用

四、Sensor + 镜头光学匹配

三个核心匹配公式，确保光学链路正确闭合

1

焦距换算 — 确定视角

根据需要的水平 FOV 反推焦距：

f = Sensor宽度 / (2 × tan(FOV/2))

例：SC231AI (1/2.9", 宽~5.3mm)，需要 90° FOV → f = 5.3/(2×tan45°) ≈ 2.65mm → 选 2.8mm 镜头 ✓

2

像素密度 vs 镜头分辨率

镜头的极限分辨率（MTF）必须匹配 Sensor 的像素密度，否则浪费 Sensor 分辨率：

Nyquist 频率 = 1000 / (2 × 像素尺寸μm) lp/mm

Sensor像素尺寸Nyquist 频率镜头 MTF 要求难度
SC223A2.9μm172 lp/mmMTF > 0.3 @172lp/mm低
SC231AI / GC2053 / OS02G102.8μm179 lp/mmMTF > 0.3 @179lp/mm中等
SC2337P / GC20832.7μm185 lp/mmMTF > 0.3 @185lp/mm较高

SC223A 的 2.9μm 大像素对镜头要求最低——配普通 3G1P 镜头即可发挥 Sensor 性能。

3

像面覆盖确认

镜头像面 ≥ Sensor 光学尺寸，且 CRA 匹配，是选型的最低可接受条件：

匹配项要求不满足的后果
像面尺寸镜头像面 ≥ Sensor 光学尺寸四角暗角 / Vignette
CRA镜头出射角 ≤ Sensor 入射角边缘亮度衰减 / Lens Shading
