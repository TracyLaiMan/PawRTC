音频基础知识体系

声音物理

dB SPL · 频率 · 波长 · 响度

声音是气压波动。SPL (声压级) 以 dB 度量, 基准 20μPa=0dB。人耳可闻 20Hz~20kHz。A-weighting 模拟人耳频率响应。距离每翻倍, SPL 衰减 6dB。

Mic (麦克风)

灵敏度 · SNR · 指向性 · 类型

灵敏度 = 给定 SPL 下的输出电压 (mV/Pa 或 dBV/Pa)。SNR = 94dB SPL 信号与自噪声的差值。指向性 = 全向/心形/8字形。ECM/MEMS/动圈三大类型。

Speaker (扬声器)

频响 · THD · 阻抗 · 分频

频响 = 各频率的输出幅度 (理想 ±3dB)。THD 总谐波失真 — 2 次谐波温暖, 3 次刺耳。阻抗 随频率变化, 最低点不可低于额定 80%。

1. 声音物理基础 — 频率 · 振幅 · 分贝 · 响度

声音是气压波动 · SPL = 20×log₁₀(P/P₀) · A-weighting 模拟人耳 · 距离每翻倍 -6dB

声音的本质

气压的周期性波动 — 纵波

声音是在弹性介质 (空气/水/固体) 中传播的机械纵波。声源振动 → 压缩/稀疏周围空气分子 → 波动向外传播。
频率 f (Hz): 每秒振动次数 — 决定音高。
振幅 A: 气压波动的幅度 — 决定响度。
波长 λ = c/f: 空气中声速 c ≈ 343 m/s (20°C)。100Hz → λ=3.43m, 1kHz → λ=0.343m, 10kHz → λ=0.034m。

分贝 (dB) 体系

对数单位 — 匹配人耳的非线性感知

dB SPL (声压级): Lp = 20·log₁₀(P/P₀), P₀ = 20 μPa (人耳听阈 @1kHz)。
关键阈值: 0dB=听阈 · 30dB=耳语 · 60dB=交谈 · 85dB=长期暴露限 · 100dB=嘈杂 · 120dB=痛阈 · 140dB=喷气机。
A-weighting (dBA): 在 1kHz 以下和 6kHz 以上衰减 → 模拟人耳频率响应。噪声测量必用。
C-weighting (dBC): 更平坦 — 用于高峰值测量 (音乐会/工业噪声)。

距离衰减与响度

距离每翻倍 → -6dB · 10dB↑ = 感知响度翻倍

平方反比定律: 点声源在自由场中, 距离翻倍 → 声压减半 → -6 dB SPL。
响度 (Loudness) vs SPL: SPL 是物理量, 响度是主观感知:
+10 dB SPL ≈ 感知响度翻倍 (需要 10× 功率)。
+3 dB SPL ≈ 刚好可分辨的响度变化 (需要 2× 功率)。
等响曲线 (Fletcher-Munson): 人耳对 2-5kHz 最敏感 (共振腔), 低频/高频需更高 SPL 才感知同等响度。

2. 麦克风 (Microphone) — 灵敏度 · SNR · 指向性 · 类型

将声波转换为电信号 · 三大类型 (ECM/MEMS/动圈) · 关键参数解读 · 选型核心考量

灵敏度 (Sensitivity)

给定 SPL → 输出电压 · mV/Pa 或 dBV/Pa

定义: 1kHz、94dB SPL (即 1Pa) 声压下, 麦克风的开路输出电压。
单位换算: dBV/Pa = 20·log₁₀(mV/Pa / 1000)
例: -38 dBV/Pa = 12.6 mV/Pa · -24 dBV/Pa = 63 mV/Pa (更灵敏!)
负值解读: 绝对值越小 = 越灵敏。−24dB > −38dB > −44dB。
选型: 安静环境用高灵敏度 (捕捉细节) · 嘈杂环境用低灵敏度 (避免削波)。

SNR & 动态范围

SNR = 94dB信号 − 自噪声 · Dynamic Range = AOP − Noise

自噪声 (Self-Noise / Noise Floor): 麦克风自身的电噪声 (等效 SPL in dBA)。低端 29dBA · 高端 14dBA。
SNR = 94 - SelfNoise(dBA): 典型值 65~80dBA。值越大 → 录音越"干净"。
AOP (Acoustic Overload Point): 失真 >1% (或10%) THD 时的 SPL。典型 110~130dB。
动态范围 = AOP - Noise Floor: 例: 110dB − 14dB = 96dB — 决定了能同时录制的最小声和最大声的跨度。

指向性 (Polar Pattern)

全向 · 心形 · 8字形 · 枪型

全向 (Omni): 360° 均匀 — 无近讲效应, 录音自然。会议/测量。
心形 (Cardioid): 正面敏感, 后方抑制 — 有近讲效应 (近距离低频增强)。人声/乐器。
8字形 (Figure-8/Bidirectional): 前+后, 两侧抑制 — 立体声/降噪 (主动抵消侧向噪声)。
超心形/枪型 (Shotgun): 极窄前方 — 影视同期声/远距离拾音。
近讲效应 (Proximity Effect): 只有压力梯度式 (心形/8字) 有 — 距离 < 30cm 低频提升 6-10dB。

类型原理灵敏度SNR典型用途
ECM (驻极体)预充电驻极体膜片 + FET 阻抗变换器−44~−24 dBV/Pa60~70 dBA消费电子 · 耳机 · IPC

MEMS vs ECM — IPC 产品 Mic 选型决策
MEMS 优势: ① 尺寸极小 (3.76×2.95mm) ② 高 SNR (>66dBA) ③ 数字输出 (PDM/I2S), 省去 ADC ④ 温湿度可靠性好 ⑤ 一致性好 (灵敏度偏差 ±1dB)。
ECM 优势: ① 成本极低 ② 多种指向性可选 ③ 高 AOP (承受高声压)。
IPC 推荐: 模拟 MEMS (高 SNR, 小尺寸) 或 数字 MEMS (直接 I2S, 省 codec ADC, 抗干扰强)。

3. 扬声器 (Speaker) — 频响 · THD · 阻抗 · 分频

电信号 → 声波 · 频响 ±3dB 标准 · 谐波失真 (2nd 温暖 vs 3rd 刺耳) · 分频器 · 灵敏度与功率

频率响应 (Frequency Response)

各频率下输出 SPL 的平坦度 · ±3dB 为优质

测量方式: 输入扫频信号 (20Hz~20kHz) → 测量轴线上 1m 处的 SPL。
±3dB 标准: 理想响应在此容差内 → "平坦"。专业监听音箱: 40Hz~20kHz ±3dB。消费级可能 ±5dB 以上。
低频滚降 (Bass Roll-off): 小型箱体低于某一频率 (f3) 后每倍频程 -12dB (封闭箱) 或 -24dB (倒相箱)。
高音延伸: 钛膜/铝带高音可至 30kHz+ (Hi-Res Audio)。

THD 谐波失真 — 读懂失真

THD = √(ΣV_harm²)/V_fund · 关键不是总数值, 是成分!

定义: THD% = 100 × √(H₂²+H₃²+...)/H₁
2 次谐波 (H₂): 八度音程 — 听起来温暖、饱满。胆机/黑胶大量产生, 主观认为是"好听"的失真。
3 次谐波 (H₃): 八度+五度 — 听起来刺耳、粗糙。过载/削波产生, 几乎总是不好听。
关键洞见: THD=1% 但全是 H₂ → 好听。THD=0.5% 但 H₃ 主导 → 难听。THD 数值不能单独评判音质。

阻抗 (Impedance)

随频率变化的交流电阻 · 最低值 ≥ 额定 80%

额定阻抗: 3.2Ω / 4Ω / 6Ω / 8Ω (最常见)。这是"标称值", 实际阻抗随频率剧烈变化。
最低阻抗 (Zmin): 通常在 200Hz 附近 (低音谐振点) 或高频段。按 IEC 标准, Zmin 应 ≥ 额定值×80% (8Ω→6.4Ω)。
与功放匹配: 阻抗过低 → 功放过流 → 保护/烧毁。两只 8Ω 并联 = 4Ω — 需确认功放能驱动。
灵敏度: 1W@1m 的 SPL。87dB(低效) → 93dB(高效)。差 3dB = 需要2倍功放功率达到同响度。

分频器 (Crossover)

分割频谱 → 低音/中音/高音各司其职

低频 (Woofer): 20Hz~2kHz · 大口径 (8-15") · 大冲程。
中频 (Midrange): 400Hz~5kHz · 关键频段 — 人声/乐器主体。
高频 (Tweeter): 2kHz~20kHz+ · 球顶/带式/AMT。
分频点 (Crossover Freq): 2-way ~2kHz · 3-way ~500Hz+3.5kHz。
主动分频: 分频→独立功放→每单元 (bi-amp/tri-amp)。优势: 低失真/高效率。

THD %相对于基频品质典型场景

4. 室内声学 — 混响 RT60 · 吸收 · 回声

Sabine 公式: T₆₀ = 0.161V/A · 吸声系数 α · 不同房间的理想 RT60 · 近讲/远场判据

RT60 — 混响时间

声压衰减 60dB 所需的时间 (秒)

Sabine 公式: T₆₀ = 0.161 × V / A
V=房间体积(m³), A=总吸声量 (Sabins = Σ αᵢ×Sᵢ)。
物理直觉: 房间越大 (V↑) → 混响越长。吸声越多 (A↑) → 混响越短。
Eyring 修正 (α>0.3 时): T₆₀ = 0.161×V / (−S×ln(1−ᾱ)) → 高吸收率时更准确。
混响 vs 回声: 混响 = 无数反射叠加的连续衰减 · 回声 = 可分辨的离散反射 (延时 >50ms, 强度 > 直达声 -10dB)。

吸声系数 α (0~1)

每种材料在每个频率的吸声能力不同

α=0: 完全反射 (混凝土) · α=1: 完全吸收 (开口窗)。
频率依赖性: 多孔材料 (吸音棉) → 高频 α 高 (0.8+), 低频 α 低 (0.1)。
共振吸收: 薄板/膜共振器 → 针对特定低频 (如 125Hz) 吸收。
典型材料: 厚窗帘 0.4~0.75 · 地毯 0.1~0.6 · 吸音板 0.5~0.9 · 玻璃 0.03~0.05。
RT60 优化方向: 短 RT(0.2~0.6s) → 录音棚/控制室 · 中 RT(0.8~1.5s) → 教室/剧院 · 长 RT(2~4s) → 音乐厅/教堂。

5. 数字音频 — 采样 · 量化 · 位深 · Nyquist

模拟→数字的关键概念 · 采样率与混叠 · 位深与动态范围 · dBFS · 抖动 (Dither)

采样 (Sampling)

连续信号 → 离散时间点 · Nyquist: fs ≥ 2×fmax

采样定理: 要无失真重建信号, 采样频率必须 ≥ 2×信号最高频率。
CD 音质: 44.1kHz → 可完整采集 0~22.05kHz (覆盖人耳上限 20kHz)。
48kHz: 视频/专业音频标准 — 与 24fps 视频帧对齐 (48k/24=2000 samples/frame)。
96kHz/192kHz: Hi-Res — 给抗混叠滤波器更平缓的过渡带, 减少相位失真。
混叠 (Aliasing): fs < 2×fmax → 高频折叠为假低频 → 不可逆失真。

量化 (Quantization)

连续幅度 → 离散台阶 · 位深决定动态范围

位深: 每采样的 bit 数 — 决定了动态范围 (DR)。
DR ≈ 6.02×N + 1.76 dB: 16bit=98dB · 24bit=146dB。
量化噪声: 量化的舍入误差, 均匀分布在 −½LSB~+½LSB → 信噪比 SNR=DR。
dBFS (dB Full Scale): 数字域基准 — 0 dBFS = 最大可表示幅值 (全 1)。所有值 ≤0 (负数)。
0 dBFS 对应的模拟电平: 由 DAC/ADC 的参考电压 (Vref) 决定 — 专业设备 +4dBu=−18dBFS (常见对应)。

抖动 (Dither)

在量化前加微小噪声 — 消除量化失真

问题: 纯正弦波在低位深时, 量化误差与信号相关 → 产生可闻的谐波失真 (颗粒感)。
解决: 加随机噪声 (dither) (幅度 ~1 LSB) → 量化误差变成不相关的白噪声 → 谐波失真消除, 换来轻微的本底噪声提升。
噪声整形 (Noise Shaping): 将 dither 噪声的频谱推向高频 (人耳不敏感) → 可闻频段内 SNR 可提升 10-20dB。

Mic 选型: MEMS 模拟 (−38dBV, SNR>66dBA, AOP>120dB) 或 MEMS 数字 (I2S/PDM, 省 codec ADC)。
Mic 增益: 从 Mic 输出到 ADC 输入的模拟增益 — 根据目标 SPL 范围和 ADC 满量程 (通常 1Vrms=0dBFS) 计算。
Speaker 选型: 微型动圈 (IPC 内置) 频响通常 300Hz~8kHz ±10dB — 注重语音清晰度 (2-4kHz 峰值)。
腔体设计: Mic 密封 (避免腔体共振) · Speaker 后腔容积决定低频延伸。
RT60 考量: 小空间 (IPC 安装位置) RT60 < 0.3s → AEC 更容易收敛 (混响尾短)。
